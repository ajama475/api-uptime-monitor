# API Uptime Monitor

A Go-based service for monitoring API and website uptime, storing status history, and supporting alerting.

## Goals
- Monitor endpoints on a schedule
- Track uptime and downtime history
- Return health and status data through an API
- Serve as a learning project with startup potential

## Tech Stack
- Go
- PostgreSQL
- Redis later if needed

## Planned Features
- Add endpoints to monitor
- Run scheduled health checks
- Store response status and latency
- View uptime history
- Alert on failures
